SCILHS Data Set Handler

This plugin fulfills a SHRINE-initiated SCILHS data request by creating a table-based CSV file with aggregated patient data for a list of patients, one row per patient. To begin, drag and drop a Flagged Query containing a patient set to the Patient(s) box below. Any concepts found in the query will be automatically added. Each concept in the list will be a column in the resulting data set. The Aggregation Option will be automatically selected based on the request, and reflect how the column(s) of data will be aggregated.
